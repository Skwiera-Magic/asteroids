# Asteroids

A classic arcade-style space shooter built with Python and Pygame.

## How to Play

1. **Download:** Grab the latest release from the `dist` folder.
2. **Run:** Double-click `main.exe` (ensure the `sounds` folder is in the same directory!).
3. **Controls:**
    * **Rotate:** `A` and `D` or Left/Right arrows
    * **Thrust:** `W` or Up arrow
    * **Shoot:** `Spacebar`

## Features

* Vector-based movement and rotation.
* Sound effects for shooting and explosions.
* Scaling difficulty as asteroids break into smaller pieces.

## Development

This game was built as part of the [Boot.dev](https://www.boot.dev) curriculum to learn Python and Object-Oriented Programming.

### Setup for Developers

If you want to run the code from source:
1. Install Python 3.10+
2. Install Pygame: `pip install pygame`
3. Run: `python3 main.py`
